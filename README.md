# Bash:

## How to install

    git clone git://github.com/ariabov/dots.git ~
    echo "source dots/*" >> ~/.bash_profile
    # Restart terminal
    aa

## Development workflow:

1. Make a change
2. ./update-aliases.sh
3. Restart terminal
